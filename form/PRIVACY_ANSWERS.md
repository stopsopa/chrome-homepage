# Privacy Tab Answers

Here are the exact answers to copy and paste into the Chrome Web Store Developer Dashboard's **Privacy** tab.

**Single purpose description**
```text
A lightweight utility that allows users to seamlessly redirect their new tab page to a custom URL of their choice.
```

**Are you using remote code?**
```text
No, I am not using Remote code
```

**Justification** *(If it asks for one)*
```text
The extension operates 100% locally and does not fetch or execute any external scripts or remote code.
```

**Data usage (What user data do you plan to collect...?)**
> *Action required: Leave ALL checkboxes UNCHECKED. (This extension does not collect any data).*

**I certify that the following disclosures are true:**
> *Action required: CHECK ALL THREE BOXES:*
> - [x] I do not sell or transfer user data to third parties...
> - [x] I do not use or transfer user data for purposes that are unrelated...
> - [x] I do not use or transfer user data to determine creditworthiness...
