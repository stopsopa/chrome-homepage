# Custom New Tab Redirect

A simple Chrome extension to redirect the new tab page to a custom URL of your choice.

Created out of a lack of trust in publicly available extensions and the realization that, in the AI era, building a custom, secure solution is simple enough. This way one should have protection from any malicious updates (supply chain attack).

## Features

- Redirects every new tab to a user-defined URL.
- Lightweight and easy to configure via the extension popup.
